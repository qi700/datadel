2007年1月第1周计算机病毒预报
下周计算机病毒预报（2007年1月1日至2007年1月7日）
病毒名称：“恶鹰变种fe”(Worm.Beagle.fe)危害程度：中高受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003未受影响的系统: Windows 3.x, Macintosh, Unix, Linux病毒危害：1. 传播性极强的隐形蠕虫2. 通过邮件发送3. 隐藏自己在系统中的信息4. 阻止反病毒软件升级，结束反病毒软件进程感染形式： 这是一个会通过邮件发送自己的蠕虫病毒，它还能通过驱动隐藏自己在系统中的信息，使用户无法察觉病毒的存在。该蠕虫正在利用互联网在全世界进行大面积传播。由于采用先进的隐形技术，可在系统隐藏自己并利用邮件服务器发送带毒邮件；同时阻止反病毒软件升级，结束反病毒软件进程。预防和清除：　  建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。
病毒名称：“武汉男生变种h”（Worm.WhBoy.h） 危害程度：中受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux病毒危害：1. 病毒运行后,会自己拷贝2. 添加注册表自启动,病毒会添加自启动项3. 寻找桌面窗口,并关闭窗口标题4. 病毒会删除文件感染形式： 这是一个感染型的蠕虫病毒,病毒会感染扩展名为exe, pif, com, src的文件,把自己附加到文件的头部并在扩展名为htm, html, asp, php, jsp, aspx的文件中添加一网址,用户一但打开了该文件,IE就会不断的在后台点击写入的网址,达到增加点击量的目的, 该病毒不但可以对用户系统进行破坏，导致大量应用软件无法使用，还能中止大量的反病毒软件进程。预防和清除： 建议电脑用户不要随便运行来历不明的文件，以免中毒受害。平时上网的时候一定要打开防病毒软件的实时监控功能。
病毒名称：“鞋匠变种fr”(Adware.licker.fr)危害程度：中受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003未受影响的系统: Windows 3.x, Macintosh, Unix, Linux病毒危害：1. 在系统中创建病毒文件2. 修改注册表，实现开机自启3. 侦听黑客指令，连接指定站点4. 弹出大量广告条幅感染形式： 这是一个广告程序，可弹出大量广告信息，并在被感染计算机上下载其它病毒。病毒运行后，在Windows目录下创建病毒文件。修改注册表，实现开机自启。自我注册为服务，服务的名称随机生成。侦听黑客指令，连接指定站点，弹出大量广告条幅，用户一旦点击带毒广告，立即在用户计算机上安装其它病毒。预防和清除：建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。
