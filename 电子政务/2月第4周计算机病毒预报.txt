2月第4周计算机病毒预报
2007年2月19日至2月25日计算机病毒预报        病毒名称：“熊猫烧香”变种CW (Worm.WhBoy.cw)    危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, Unix, Linux    病毒危害：    1. 自身复制病毒文件    2. 修改和添加注册表，实现自启动    3. 结束杀毒软件和安全软件的保护程序    4. 利用密码字典破解弱密码    5. 感染区域网内其他机器    感染形式：     该病毒是"熊猫烧香"的一个新变种病毒,它除了可以感染系统中可执行文件外,还会感染网页文件，导致系统硬盘的资料和数据文件被损坏，同时它还能通过局域网传播，感染局域网内所有计算机系统，危害极大，可能使整个局域网崩溃，无法正常运作。 该病毒运行后,它会把自身复制为ncscv32.exe病毒文件。同时修改注册表,实现随开机自动启动。它会强行结束各个杀毒软件和安全软件的保护程序,而且能够终止spoclsv.exe、nvscv32.exe、sppoolsv.exe、spo0lsv.exe等旧版变种熊猫烧香病毒的进程。此外,它会感染扩展名为exe,pif,com,src的文件,将熊猫烧香的图像粘贴上去,还会利用密码字典破解弱密码,登录并感染区域网内其他机器,造成大规模地扩散。    预防和清除：    建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便打开不明来历的可执行文件以免中毒受害。
病毒名称：“维金”变种AN（Win32.Vcing.an）     危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux    病毒危害：    1. 修改注册表，实现随开机自动启动    2. 搜索系统中的磁盘    3. 损坏硬盘的资料和数据文件    感染形式：     该病毒是一个活跃于windows平台的感染型病毒，它会感染电脑系统中所有扩展名为.exe可执行文件,可导致用户的系统硬盘的资料和数据文件被损坏。该病毒运行后,它会释放并执行lsass.exe病毒文件。同时修改注册表，实现随开机自动启动。同时，它会从c盘起搜索系统中的磁盘，并感染磁盘中的以.exe为扩展名的文件。此外,它还会感染扩展名为.htm的网页文件。    预防和清除：    建议电脑用户不要随便运行来历不明的文件，以免中毒受害。平时上网的时候一定要打开防病毒软件的实时监控功能。
