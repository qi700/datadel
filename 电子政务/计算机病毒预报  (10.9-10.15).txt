计算机病毒预报  (10.9-10.15)
病毒名称：“下载变种ha”(Worm.Downloader.ha)     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, Unix, Linux     病毒危害：     1.生成病毒文件     2.下载病毒并按其内容进行更新     3.添加收藏夹信息、弹出广告等操作     4.通过即时聊天工具自动传播     感染形式：     该病毒是一个广告蠕虫病毒。它在受感染系统中生成以下病毒文件：     %SystemRoot%\system32\PopWin.exe     %SystemRoot%\system32\PopTWin.exe     %SystemRoot%\system32\PopWin.dll     %SystemRoot%\system32\delme.bat     %SystemRoot%\system32\update.web     %SystemRoot%\system32\update1314.exe，     病毒运行后会下载病毒并按其内容进行更新、IE主页修改、添加收藏夹信息、弹出广告等操作。该病毒主要通过即时聊天工具自动传播，建议广大电脑用户在聊天时不要随便打开聊天窗口中连接信息，避免感染病毒或成为病毒感染源。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。          病毒名称：“下载者变种ds”（Troj.Downloader.ds）     危害程度：中低     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1.该病毒是一个下载木马     2.下载其他病毒并运行     3.盗取QQ及网游帐号     感染形式：     该病毒是一个下载木马。该病毒可下载其他病毒并运行，对其受感染的系统不做修改；该病毒下载病毒大多是盗取QQ及网游帐号的木马病毒。     预防和清除：     建议电脑用户不要随便运行来历不明的文件，以免中毒受害。平时上网的时候一定要打开防病毒软件的实时监控功能。          病毒名称：“海盗变种an”(Worm.Viking.an)     危害程度：中高     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, Unix, Linux     病毒危害：     1.利用网络共享进行传播的蠕虫     2.修改注册表，实现开机自启     3.侦听黑客指令，下载病毒文件     4.终止与安全相关的进程     感染形式：     这是一个利用网络共享进行传播的蠕虫。病毒运行后，在Windows目录下创建病毒和一个木马下载器。修改注册表，实现开机自启。将木马下载器注入到ieplorer.exe或explorer.exe的进程中，侦听黑客指令，利用木马下载器从黑客指定站点下载病毒文件。遍历用户计算机的C到Y驱动器，搜索并感染后缀是.exe的文件。利用空密码打开共享文件夹，并自我复制到共享文件夹下，感染所有的.exe文件，实现网络共享传播。另外，该病毒还可以终止与安全相关的进程，降低被感染计算机上的安全设置。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要轻易运行从Internet下载后未经杀毒软件处理的文件，当操纵者控制用户电脑时，就可直接导致用户的信息被泄露， 为了您系统和个人信息的安全，专家建议用户在打开一个陌生文件时，请用最新病毒库的杀软进行扫描
