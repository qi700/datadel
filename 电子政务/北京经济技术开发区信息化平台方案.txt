北京经济技术开发区信息化平台方案
北京经济技术开发区地处北京东部发展带亦庄地区（以下简称开发区），是北京唯一同时享受国家级经济技术开发区和国家级高新技术产业园区双重优惠政策的特殊经济区域。当前，开发区政务信息化建设已取得了一定的进展：一些业务管理信息系统已开发完成并投入使用，并已形成了较为完备的计算机网络系统，但相对于经济技术开发区的政务信息化总体建设目标而言，开发区当前政务信息化的发展仍处于起步阶段，主要体现为信息化建设缺乏总体规划，没有形成有效的信息资源管理机制，已经建成的各应用系统只针对单一业务需求，缺乏系统之间的有机联系与信息共享。上述情况已经无法满足建设“服务型政府”的需要，更无法实现开发区“十一五”规划中提出的“将开发区建成北京市信息化示范区，信息化综合水平保持北京市领先，完成构建首都信息社会”的目标。客户需求：本项目将要建设的开发区核心信息资源数据库，包括企业法人数据库、空间地理信息数据库、宏观经济数据库、人口数据库等。开发区区域核心信息资源数据库的目的是实现开发区政务信息数据资源的交换、共享、整合、开发及利用。在本项目中，要理顺数据源与开发区管理部门之间的关系，解决上述各个数据库的数据收集、存储、维护和现有数据的导入等问题，作为未来的开发区数据资源整合交换平台建设的基础。同时，将数据资源作为服务提供给合法的使用者，从而实现数据资源的共享和利用。由于数据库的安全和效率的需求，数据库服务系统需要做到稳定高效和具有一定灾难恢复能力。
解决方案及部署实施：
针对用户的数据库系统建设目标，结合用户的相关应用，红旗公司提供了综合的解决方案，帮助用户建立起了从底层的操作系统平台软件，到高端的HA高可用软件的一整套的部署方案。在操作系统平台上，采用了红旗 Linux DC Server 5.0 for PPC版本，充分利用红旗服务器操作系统在管理性、可用性、可靠性和扩展性上的优势。Linux DC Server 5.0 for PPC定位于企业级的网络和应用服务器。它运行在IBM 高端的PowerPC560Q架构服务器上；提供稳定的运行和业界主流的商业应用。此外，该产品还可以作为完整的 Linux 软件开发平台。作为高端的应用软件，红旗提供了高可靠的集群解决方案。对政府政务系统来说，及时和可靠的信息服务是必要的，但是软硬件的故障又是不可避免的，为减少故障带来的损失，Red Flag HA Server 5.0 for PPC为用户提供24小时昼夜不停的可靠保证，确保网络系统、网络服务、共享存储、共享文件系统、进程以及数据库能够不停息的运转。在本项目中，红旗高可用软件被应用于Oracle10g数据库等关键应用，以提升数据库系统的稳定性和可靠性，保障应用的平稳运行。项目拓扑图：

从系统的高性能和高稳定性出发，尽量降低运行中的故障风险,本项目综合使用了如下软硬件产品，共同搭建了一个高效、稳定、安全的集群系统平台：软件：操作系统：采用中科红旗的Red Flag DC Server 5.0 for PPC 版本。红旗Linux数据中心服务器可以满足系统对性能、可靠性、可用性、安全性以及成本控制等方面的要求。集群软件：采用中科红旗的Red Flag HA Cluster 5.0 for PowerPC版本，构建数据库集群。数据库系统：采用国际知名品牌Oracle Serer 10g ，这样保证数据的高可用性。web服务器系统：应用服务器软件采用weblogic8.3，保障系统的高性能服务。硬件：IBM小型机：P560Q服务器2台。磁盘阵列：IBM DS4700光纤柜。网站安全防护系统：采用天融信4324硬件防火墙系统.
效果评价：
在亦庄区政府管委会和集成商的大力支持下，中科红旗软件技术有限公司顺利实施部署了区域信息化基础平台；整合了开发区的区域信息资源；建设与完善核心电子政务应用系统；完成数据及应用系统的迁移；建设并完善了开发区信息化建设的标准体系以及电子政务规范体系。红旗软件表现出卓越的运营性能，良好的数据运行效果，和及时、高效的服务保障。
