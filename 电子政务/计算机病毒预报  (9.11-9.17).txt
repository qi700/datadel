计算机病毒预报  (9.11-9.17)
病毒名称：“下载者变种vb”(Troj.Downloader.vb)     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, Unix, Linux     病毒危害：     1.自身复制为伪系统正常文件     2.添加注册表项     3.下载其它病毒     感染形式：     这是一个Windows平台下载其它木马的病毒，病毒将自身复制为以下伪系统正常文件:%Windir/Winsvc.ex，并且添加注册表项，利用注入技术绕过网络防火墙的监视，在网络资源可用时，下载其它病毒。该病毒主要通过系统漏洞，欺骗安装方式进行传播。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。          病毒名称：“传奇密码大盗变种ci”（Troj.PSWLmir.ci）     危害程度：中低     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1.病毒主要通过网络欺骗和捆绑软件方式进行传播     2.复制自身成伪系统文件     3.将信息发送给病毒作者     4.盗取玩家账号和密码     感染形式：     该病毒为Windows平台下的专门针对于传奇网络游戏的盗号型木马，病毒运行后复制自身成伪系统正常文_svchost_.exe，然后利用钩子技术将病毒代码注入每一个进程中，发现传奇网络游戏时，病毒利用键盘钩子和鼠标钩子记录用户帐号、密码、角色装备信息，然后将信息发送给病毒作者。同时该病毒运行过程中尝试通过网络下载并运行其它病毒。病毒运行过程中实时监视并关闭相关反病毒软件。     预防和清除：     建立良好的安全习惯，关闭或删除系统中不需要的服务；很多病毒利用漏洞传播，一定要及时给系统打补丁；安装专业的防毒软件进行实时监控，平时上网的时候一定要打开防病毒软件的实时监控功能。          病毒名称：“灰鸽子2006变种jq”(Backdoor.Huigezi.2006.jq)     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, Unix, Linux     病毒危害：     1．自我复制到系统目录下     2．修改注册表，实现开机自启     3．侦听黑客指令，盗取计算机信息     4．终止某些杀毒软件和防火墙程序的进程     感染形式：     这是一个利用操作系统漏洞进行传播的后门。该病毒运行后，自我复制到系统目录下，文件名与Windows系统文件名相似，请广大用户注意识别。修改注册表，实现开机自启。利用自己的IRC客户端连接指定的IRC服务器，侦听黑客指令，盗取计算机系统信息和各种游戏的序列号，并对指定目标执行DoS攻击等。该后门利用微软漏洞、缺省或弱密码网络共享传染给其他用户计算机。另外，该病毒还终止某些杀毒软件和防火墙程序的进程，阻止用户对一些常见安全网站的访问。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要轻易运行从Internet下载后未经杀毒软件处理的文件，当操纵者控制用户电脑时，就可直接导致用户的信息被泄露， 为了您系统和个人信息的安全，专家建议用户在打开一个陌生文件时，请用最新病毒库的杀软进行扫描
