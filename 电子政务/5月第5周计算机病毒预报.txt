5月第5周计算机病毒预报
2007年5月28日至6月3日计算机病毒预报      病毒名称：“键盘终结者”变种gd (TrojanSpy.KeyLogger.gd)    危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, Unix, Linux    病毒危害：    1. 记录用户的键盘和鼠标操作    2. 窃取用户输入的机密信息    3. 以邮件的形式发送给黑客    4. 泄露用户的私密信息    感染形式：     这是一个木马病毒，采用Visual Basic编写。“键盘终结者”变种gd能够在后台秘密记录用户的键盘和鼠标操作，窃取用户输入的机密信息，并将这些信息保存为日志文件，以邮件的形式发送给黑客，从而泄露用户的私密信息，给用户带来损失。    预防和清除：    建立良好的安全习惯，不打开可疑邮件和可疑网站；建议电脑用户不要随便打开不明来历的可执行文件以免中毒受害。
病毒名称：“哲拉蒂”变种axw (I-Worm.Zhelatin.axw)    危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, Unix, Linux    病毒危害：    1. 搜索用户电脑上有效的邮件地址信息    2. 发送含有病毒附件的电子邮件    3. 占用网络资源,导致用户的电脑系统速度减慢    4. 网络带宽被严重占用甚至瘫痪等    感染形式：     这是“哲拉蒂”蠕虫家族的最新变种之一，采用Visual C++编写。病毒运行后，在Windows目录下释放病毒文件。修改系统默认的LSP（分层服务提供程序），导致一连接网络就会启动“哲拉蒂”变种axw。在后台秘密监视用户与网络通信的所有信息内容，窃取用户输入的机密信息，并通过Web邮箱发送到黑客指定站点。另外，病毒还可以从指定站点下载其它木马程序。    预防和清除：    建立良好的安全习惯，不打开可疑邮件和可疑网站；建议电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。
病毒名称：“红猫”(Win32.RedCat)    危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux    病毒危害：    1． 在电脑盘符下生成病毒文件    2． 感染扩展名为.exe的文件    3． 修改可执行的EXE文件和mp3文件    4． 替换原来图标为“粉红色的小猫”    感染形式：     该病毒是一个新爆发的感染型病毒,它会感染用户电脑上的大量的可执行的EXE文件和修改mp3文件, 以一个“粉红色的小猫”的图标替换原来图标,造成粉红小猫遍满桌的现象。此外,它还会在电脑盘符下生成病毒文件,当用户双击该盘符时便会激活病毒。该病毒运行后，感染扩展名为.exe的文件，并出现“粉红色的小猫”的图标；修改扩展名为.mp3的文件名为“.Mp三killxxx”，其中“xxx”是随机的数字。在电脑的D盘下生成setup.exe及autorun.inf病毒文件。    预防和清除：    建议电脑用户不要随便运行来历不明的文件，以免中毒受害。平时上网的时候一定要打开防病毒软件的实时监控功能。
