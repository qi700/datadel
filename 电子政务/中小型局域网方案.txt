中小型局域网方案
随着市场竞争的日趋激烈，企业为了提高竞争力，急需深化和普及其信息化策略。同时事业和政府单位组织为了提供更好的服务和更有效的管理，也在大力制定和推广其信息化战略。随着国家经济的蓬勃发展，中小型企业的数量在不断增长，目前中小规模的网络建设已经成为网络市场的热点。
作为企业网络市场的领导者，中小规模网络市场一直是3Com公司的市场重点之一。3Com针对中小规模局域网络的建设特点，基于新一代的ASIC芯片技术和先进的软件技术，设计推出了丰富的局域网络产品和解决方案。
一、3COM公司面对中小企业局域网用户的产品方案
3Com公司的局域网络核心交换产品，除了能提供支持大规模局域网络的高密度的千兆交换模块化交换机Switch4000系列外，针对中小企业局域网络的需求特点，3Com还推出SuperStack 3 Switch 4900系列基于固定端口的千兆交换机。SuperStack 3 Switch 4900系列交换机采用3Com新一代ASIC芯片技术可提供12/16/24/28等多种端口密度和多端口媒介的丰富选择，并具备L2/L3/L4层全线速的交换性能和高度的智能化功能，并且其平均每千兆端口的价格大大低于模块化核心交换设备。
在网络边缘的桌面交换方面，3Com公司同步推出了支持第四层交换技术的SuperStack 3 Switch 4400 10/100M全线速智能型交换机，开桌面交换智能化新标准之先河。它与Switch 4000系列和SuperStack 3 Switch 4900系列核心交换机相配合构成了高性能价格比的端到端的智能化网络解决方案。
另外，3Com针对注重网络性能而对于功能性要求不高的用户推出了SuperStack 3 Switch 4200系列经济型10/100M桌面交换机，它具备全线速的性能和丰富的第二层交换功能极具性价比。
与此同时，3Com公司还推出了建设网络核心的新技术理念,“XRN”弹性可扩展网络核心技术。通过XRN技术多个独立的核心交换机可以组成互为冗余的负载共享的分布式交换集群，而对外就象是一台核心交换机一样。以前用户为了解决网络核心的弹性和可扩展性问题，常会超前投入购买大规模的模块化核心交换设备。但是由于市场环境的不断变化带来的企业未来的网络规模和业务应用需求发展的不定性，同时网络新技术和产品也会层出不穷，这种超前投入的方法势必会极大的降低网络投资效率，而XRN技术为用户提供了渐进式投入、逐步按需扩展的建设网络核心的新方法。3Com的XRN技术可以广泛应用于企业局域网络核心交换设备。SuperStack 3 Switch 4900系列千兆交换机支持XRN技术，可为中小规模的局域网络用户提供可扩展的冗余的负载共享的千兆网络核心解决方案。
SuperStack 3 Switch 4900系列千兆核心交换机、XRN技术、SuperStack 3 Switch 4400 智能型桌面交换机和SuperStack 3 Switch 4200经济型桌面交换机，使得--高性能、高可靠性和端到端的智能化网络，不再意味着昂贵的网络投资，3Com为中小规模网络用户提供了先进而实用的高性能价格比的局域网络解决方案。
这类中小型企业网络的特点是：用户需要高性能而低成本，具备一定的网络服务、控制能力和有效的网络管理功能。总体上来说可分为两个层次，我们在下文中会详细说明。
1. 千兆核心层
在3COM公司的中小型企业网络方案中，千兆核心层网络采用SuperStack 3 Switch 4900千兆以太网络交换机，根据具体情况可采用光纤或双绞线作为千兆传输媒介。
SuperStack 3 Switch 4900系列交换机是适用于中小规模千兆局域网络的核心交换机和大规模网络的千兆汇聚交换机，如图1所示。
采用3Com新一代ASIC芯片技术可提供12/16/24/28等多种端口密度和多端口媒介的丰富选择，并具备L2/L3/L4层全线速的交换性能和高度的智能化功能，与3ComXRN技术相配合，更能提供冗余负载共享的可扩展局域网络核心。其平均每千兆端口的价格大大低于模块化核心交换设备。
该系列包括四款产品
（1）SuperStack 3 Switch 4900交换机(两款)
SuperStack 3 Switch 4900交换机分为两种型号1000BASE-T(RJ-45)和1000BASE-SX(MT-RJ)，它们分别适用于廉价的5类双绞线和多模光纤传输介质。前面板可提供12个固定的千兆以太网端口。还可在背板扩展4个千兆端口(1000BASE-T/1000BASE-SX/1000BASE-LX/1000BASE-GBIC可选)。可同时支持铜缆(五类双绞线)、单模和多模光纤的千兆以太网。这样，SuperStack 3 4900交换机可提供共16个千兆端口，32Gbps的无阻塞第二/三层交换能力，其IP访问控制列表功能可增强网络的安全控制能力，其自动配置功能更使网络的安装与管理变得简单而快捷。它既可以独立作为中小规模局域网络的千兆核心，或者应用在分布式网络作为以Switch 4000系列形成的千兆网络骨干的汇聚层交换核心，以及为服务器群提供千兆高密度接入。
（2）SupeStack 3 Switch 4924/4950交换机
SS3 4924/4950交换机基于3Com全新的56Gbps多层交换ASIC体系。可提供28千兆端口，四千一百七十万包的无阻塞、零丢包率的第二/三/四层交换。除支持通常的第三层交换功能外，还提供多层质量服务功能，IP访问控制列表安全控制功能等。
SS3 4924交换机提供24个10/100/1000端口并可通过扩展槽提供4个各种介质的千兆上连端口。基于高性能的多层交换体系，4924可作为基于双绞线系统的中、小规模网络的核心交换设备，以及服务器群和千兆汇聚接入多层交换设备。
SS34950交换机提供混合双绞线、光纤和GBIC的24个固光纤端口以及6个GBIC端口1000BASE-SX，1000BASE-LX定千兆端口。12个10/100/1000端口，6个1000BASE-SX多模，1000BASE-LH70可选，其中LH70可提供70公里的城域千兆连接。也可通过扩展槽提供4个各种介质的千兆上连端口。4950以其高性能、低拥有成本，灵活的介质支持成为中、小规模网络核心的理想选择。
2. 网络边缘层
网络边缘层可根据具体网络应用需求采用SuperStacak 3 Switch 4400堆叠系统/4200/4300独立交换系统。服务器入网可采用3Com 3C996T双绞线千兆服务器网卡。
（1）SuperStack 3 Switch 4200
SuperStack 3 Switch 4200系列产品是3Com公司采用最新的技术推出的全固定配置、第二层10/100M交换机。它以极为合理的价格提供：全线速性能、丰富的第二层交换功能、真正的即插即用、简单堆叠技术，为用户提供了又一个高性价比的桌面接入解决方案。其快速生成树、链路聚合和冗余电源支持使得该桌面接入解决方案更富于弹性和高可靠性。其产品外观和堆叠连接图示如图2左、右图所示。
SuperStack 3 Switch 4200系列目前包括两款产品：
SuperStack 3 Switch 4226T：提供24个10/100M和2个10/100/1000M固定端口
SuperStack 3 Switch 4250T：提供48个10/100M和2个10/100/1000M固定端口
（2）．SuperStack 3 Switch 4300交换机
SuperStack 3 Switch 4300交换机也是3Com公司一款具有48个快速以太网端口，并且可扩充4个高速端口的高密度桌面交换机，每端口价格几乎相当于市场上同类产品价格的一半，并提供高密度交换性能。产品外观、堆叠连接和模块示意图如图3所示。
SuperStack 3 Switch 4400系列产品是基于新一代ASIC芯片技术的可堆叠的全线速智能化的10/100M交换机。它的特点主要体现在三个方面：先进的堆叠技术可使桌面接入方案更灵活、更可靠和更简单；基于第四层交换技术的网络控制能力使得网络具备端到端的智能化；只相当于过去第二层10/100M交换机的价格，性价比非常好。它与Switch 4000系列核心交换机或SuperStack 3 Switch 4900系列交换机构成的高速千兆骨干相配合提供高性能高密度桌面接入，并组合成端到端的智能化网络。
SuperStack 3 Switch 4400系列包括24个10/100M交换端口或440048个10/100M交换端口以及4400SE三款产品。每款均还提供两个可扩展槽，可提供两个千兆上连端口或一个千兆上连端口和两个堆叠端口。所有端口可全线速交换。
SuperStack 3 Switch 4400采用3Com新一代堆叠技术，最多可扩展到200个网络交换端口。该堆叠系统可有效的解决了传统堆叠系统的单点故障问题，具备'弹性堆叠'和'分布式管理'功能。弹性堆叠可使堆叠系统中任意单元发生故障时并不影响堆叠中其他设备的正常运行，并可热更换故障单元。该堆叠系统采用单一网管地址管理，分布式管理可使堆叠系统中任意单元发生故障时，并不影响其他单元的网络管理。从而实现堆叠系统的高可靠性。其独特的"跨越堆叠单元的链路聚合"技术，可在单一堆叠系统内提供多达4条互为冗余、负载共享的千兆上连线路，有效的解决了高密度堆叠系统的上连带宽瓶颈问题。
SuperStack 3 Switch 4400具备高度智能化功能。支持802.1P和IP Different Service标记优先级识别，用户也可根据数据包内的协议类型、IP地址、端口号等信息自主定义优先级别，并可实现以上两种方式的影射。这样4400可从网络第二/三/四层为企业各种网络应用提供质量服务以及安全控制，从而使企业网络从核心到边缘真正实现端到端的质量服务和安全策略。
SuperStack 3 Switch 4400SE在硬件结构上与440024端口交换机相一致，它支持第二层交换功能，并可通过专门的系统软件(需要付费)升级到4400。4400SE非常适合短期内网络投资资金比较紧张或暂时用不上智能交换功能，而同时又考虑到未来业务发展的高要求并要考虑未来对现有网络投资保护的企业网络用户。
二、3COM的中小企业局域网方案
下面将以SuperStack 3 Switch 4900系列交换机和SuperStack 3 Switch 4400和4200系列交换机以及XRN技术为主，推荐3Com面向中小规模网络用户的局域网络解决方案。
1. 平面型局域网络组网方案
此种网络网络结构如图4所示。本案尤其适用于网络结构化布线采用纯双绞线系统的平面型局域网络用户。另外，可通过SuperStack 3 Switch 4900系列的扩展槽，扩展光纤上连端口，完成与其他网络的互连。
2. 楼宇型局域网络组网方案
此种网络结构如图5所示。本案适用于网络结构化布线采用较多的光纤作为垂直主干的用户。对于采用SuperStack 3 Switch 4900SX用户，可通过扩展卡提供1000BASE-T的千兆服务器连接。
3. XRN技术组网方案
此种网络结构如图6所示。采用XRN技术目前可使得网络核心扩展到48千兆端口，可以满足大部分局域网络对千兆端口密度的要求，而且将来可以梯次逐步扩展。
采用XRN技术构建的弹性可扩展网络核心方案与传统的双冗余网络核心方案相比较具备如下优势：
·　高可扩展性
XRN技术实际上是一种“集群”技术，它可把多台核心交换设备互连成一个整体，象一台交换机一样的工作，其灵活的网络核心的可扩展性是传统方案所无法比拟的。
·　全性能高可靠性
传统的双冗余方案，必有一部分主干接口和线路是用于备份的，只有当主设备和线路发生故障时，它们才投入使用，用户的投入只获得了网络的可靠性。而XRN方案所有的主干接口和线路均可以同时使用，使得网络主干性能加倍，用户的投入同时既获得了可靠性和性能，投资效率极大提高。
·　易于实施与管理
传统的双冗余方案，采用多种混合技术实现，网络设置非常复杂，而XRN技术则管理和实施非常简单，有效的降低了网络运营维护成本和管理风险。
