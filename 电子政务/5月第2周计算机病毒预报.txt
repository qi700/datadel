5月第2周计算机病毒预报
2007年5月7日至5月13日计算机病毒预报      病毒名称：“Adodb陷阱”变种g (TrojanDownloader.XmlAdodb.g)    危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, Unix, Linux    病毒危害：    1. 连接指定站点    2. 自动下载指定的木马程序    3. 窃取被感染计算机上的QQ用户名和密码    4. 发送到黑客指定的邮箱中    感染形式：     这是一个利用微软漏洞进行传播的木马下载器，由Delphi语言编写，并经过UPX加壳处理。病毒运行后，连接指定站点，在后台自动下载指定的木马程序，窃取被感染计算机上的QQ用户名和密码，并发送到黑客指定的邮箱中，给用户带来损失。    预防和清除：    建立良好的安全习惯，不打开可疑邮件和可疑网站；建议电脑用户不要随便打开不明来历的可执行文件以免中毒受害。
病毒名称：“传奇窃贼”变种gsy (Trojan.PSW.LMir.gsy)    危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, Unix, Linux    病毒危害：    1. 修改注册表，实现开机自启    2. 发送伪造的ARP数据包    3. 造成全网计算机瘫痪    4. 盗取整个局域网里传奇游戏玩家的帐号和密码    感染形式：     这是“传奇窃贼”木马家族的最新变种之一，由Delphi工具编写。“传奇窃贼”变种gsy运行后，在%WinDir%目录下面释放出病毒文件kb8964225.log。修改注册表，实现开机自启动。发送伪造的ARP数据包，使被感染的计算机伪装成网关，致使全网收发的数据包都会发向被感染的计算机，造成全网计算机瘫痪。盗取整个局域网里传奇游戏玩家的帐号和密码，给玩家带来极大的损失。    预防和清除：    建立良好的安全习惯，不打开可疑邮件和可疑网站；建议电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。
病毒名称：“敲诈者”变种CM(Win32.Adware.Virtumonde.cm)    危害程度：中    受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003    未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux    病毒危害：    1． 弹出广告程序    2． 谎报系统出现漏洞    3． 搜索的关键字及电脑上的相关有效信息    4． 占用系统资源，导致死机    感染形式：     这是以一个广告程序的形式出现,它会在受感染的电脑系统中不断地弹出带有谎报系统出现漏洞的文字窗口,诱骗用户购买该修复程序。此外,它还会收集用户在google搜索引擎搜索的关键字及电脑上的相关有效信息,并将这些信息发送到指定的网址。该敲诈窗口不但会给用户造成恐慌,而且占用系统资源,可能会导致死机等现象。    预防和清除：    建议用户多留意相关安全资讯,提高互联网的安全知识,避免病毒给您带来的损失。
