工业和信息化部组建
3月15日，十一届全国人大一次会议通过了关于国务院机构改革方案的决定，根据改革方案，国务院将组建工业和信息化部。
根据改革方案，工业和信息化部组建成立后，将不再保留信息产业部、国防科工委及国信办。将国家发改委工业行业管理职责、国防科工委除核电管理以外职责、信息产业部和国信办的职责，整合划入该部。
国务院秘书长华建敏表示，我国正处于工业化加速发展的重要阶段，走新型工业化道路，推进信息化和工业化融合，推进高新技术与传统工业改造结合，促进工业由大变强，是当前和今后一个时期的重要任务。
他表示，目前，工业行业管理由国家发展和改革委员会、国防科学技术工业委员会、信息产业部分别负责，管理分散，不利于工业的协调发展。为加强整体规划和统筹协调，有必要对相关职责进行整合，组建工业和信息化部。
新成立的工业和信息化部的主要职责将是，拟订并组织实施工业行业规划、产业政策和标准，监测工业行业日常运行，推动重大技术装备发展和自主创新，管理通信业，指导推进信息化建设，协调维护国家信息安全等。
根据国务院的任命，原国家安监局局长李毅中当选工业和信息化部首任部长。王旭东、奚国华（行情,资讯,评论）、苗圩、陈求发、娄勤俭、欧新黔、杨学山等7人为副部长。

网络交易将出新规
日前，商务部信息化推进司副司长聂林海透露，商务部计划在今年6月份发布《网上商业数据保护办法》，这将是中国首部网上交易相关法规。据悉，目前该办法已经完稿，正处于向相关部门征求意见环节。另外，酝酿近一年时间的《网上交易管理办法》也已进入起草阶段，预计将在今年年内出台。2007年3月，商务部曾发布实施《关于网上交易的指导意见》。
治理垃圾短信有标准
据悉，为进一步治理垃圾短信，信息产业部将出台一系列新措施。其中包括从技术上给用户提供垃圾短信管理平台。相关部门已研究制定了相应的设备和业务技术标准，希望从电信设备和电信业务入手给用户提供自我设置的垃圾短信管理平台。目前，该标准已进入审批阶段。此外，政府也鼓励移动通信运营商采取新措施新业务，以遏制垃圾短信。
北京无线城市现雏形
据悉，北京无线城市一期工程已于3月底完成，5月进入试运行。负责北京市“无线城市”项目的所有基站建设与线路铺设为中电华通公司，网络建设主要基于Wi-Fi和WiMax技术。北京无线城市项目计划2008年上半年完成奥运会热点地区的无线网络建设，重点覆盖奥林匹克中心区以及其他奥运场馆周边地区；以及重要交通枢纽、重要旅游景点、主要公共场所等。
微软发布企业级新“旗舰”
3月13日，微软公司发布了其新一代企业级应用平台与开发技术的3大旗舰级产品：Windows Server 2008、Visual Studio 2008和SQL Server 2008。
微软表示，此次微软发布的3大产品将有助于企业实现“动态IT”愿景。随着Windows Server 2008、Visual Studio 2008和SQL Server 2008的推出，IT专业人士和开发人员将有更多的时间进行技术创新，公司将建立起自动管理现有系统的IT基础架构，使管理变得更为简单。更重要的是，它们将为根据商业环境的变化和战略目标自动调整计算资源的IT基础架构打下基础。
微软公司高级副总裁、主管服务器和工具业务的鲍勃·穆格利亚(Bob Muglia)称，这3款企业级产品，集合了微软在管理、安全、协作等方面的理念，为IT专业人员和开发人员能够快速轻松地创建出动态系统奠定了重要的基础。
在新产品发布之前，已经有众多知名企业先期部署了Windows Server 2008。 作为最终用户先期体验计划的一部分，Windows Server 2008 (RDP)为客户提供了早期体验的机会。阿里巴巴、广东省信息中心、辽宁移动、天狮集团等企业都参加了这一计划。Visual Studio 2008中文版已于2008年2月1日上市销售，Windows Server 2008中文版将于2008年5月1日投入市场。SQL Server 2008完整功能的2月份技术预览(CTP)已发布，普遍供应则要等到2008年下半年的晚些时候。

“经济衰退会对首席信息官(CIO)扮演的角色产生影响，但并不是大问题。同时，这要求CIO具备更多的能力，包括在经济困难时期更好地配置资源的能力。”
——CIO领导力中心执行总监哈维·科佩尔(Harvey Koeppel)在接受《信息周刊》采访时表示，该中心由国际商业机器公司(IBM)发起成立。目前全球经济出现动荡，经济前景并不乐观，科佩尔提醒企业IT主管，越是面临经济不景气，越要把资源用在刀刃上，特别是用在留住客户上。

观点
“云计算”风起云涌
Forrester研究公司称，云计算有望成为一项革命性的技术，虽然现在还不足以满足大众市场的需求，但无疑拥有改变现有格局的潜力。云计算可以让用户绕过传统的企业IT部门，以更快的速度获得服务。云计算同现有的托管服务有着很大的区别：现有托管服务的技术基础设施主要针对托管几家客户而优化；云计算服务提供商则广泛应用了虚拟化技术和网格计算软件。云计算服务提供商包括亚马逊、Akamai科技、Joyent、Mosso软件等。
企业规模决定IT采购偏好
国际数据公司(IDC)最新调查显示，在采购IT服务时，超过64%的受访中小企业更乐意同当地的系统集成商或服务供应商合作。对于小企业而言，这种偏好更加明显。另一方面，中型企业更偏向于与技术所有者合作，且与跨国公司或国际服务供应商结成合作伙伴的可能性更大。在确定IT服务供应商时，中小企业最为重视价格、本地化知识和市场经验，而中型企业将服务商对其的关注程度以及响应时间作为重要的决定性因素。
场总监李永起说。IT应用对于绿色节能的要求早已迫在眉睫，据Gartner预计，50%的现有数据中心的电力和冷却能力到2008年将难以满足高密度计算系统的需求。为此，Sun提出了Eco理念，帮助企业基于环保的基础上实现高能效的IT基础设施创新。
