计算机病毒预报  (8.14-8.20)
病毒名称：“传播者”(Worm.Sower)     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, Unix, Linux     病毒危害：     1.通过多种方式传播     2.生成病毒文件     3.添加注册表项     4.结束大量反病毒软件进程，降低系统的安全性     感染形式：     这是一个可以通过多种方式传播的蠕虫病毒。该病毒生成病毒文件，添加注册表项。该病毒能结束大量反病毒软件进程，降低系统的安全性。并且该病毒可以通过可移动磁盘，p2p文件共享软件，通过弱口令攻击，以及邮件多种方式进行传播。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。          病毒名称：“卡咳虫”（Backdoor.Agent）     危害程度：中低     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1.我复制到系统盘根目录下     2.修改注册表，实现开机自启     3.盗取被感染计算机上的机密信息     感染形式：     这是一个盗取被感染计算机上机密信息的蠕虫。病毒运行后，自我复制到系统盘根目录下。修改注册表，实现开机自启。侦听黑客指令，盗取被感染计算机上的机密信息（例如，邮箱的用户名和密码，剪贴板数据等），并将盗取的信息保存在黑客指定的文件里。     预防和清除：     建立良好的安全习惯，关闭或删除系统中不需要的服务；很多病毒利用漏洞传播，一定要及时给系统打补丁；安装专业的防毒软件进行实时监控，平时上网的时候一定要打开防病毒软件的实时监控功能。          病毒名称：“穆尼亚”（Win32.Munia.a）     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1．在系统目录下释放病毒文件     2．将病毒文件注入到这些程序中     3．导致其中一些软件出现异常     4．修改系统内核入口指令     感染形式：     该病毒会在系统目录下释放病毒文件，然后搜索电脑上所有运行的程序，把病毒文件注入到这些程序中。病毒每隔1秒钟搜索一次系统目录，如果被删除则重新复制自己，以保护自己不被杀毒软件查杀。该病毒会释放窃取网络游戏“传奇”的模块，使玩家的账号、密码等资料丢失。跟其它的盗号病毒不同的是，该病毒会感染中毒电脑上运行的所有程序文件，并可能导致其中一些软件出现异常。并且，该病毒还会修改系统内核入口指令，使用户在中毒电脑上运行任何程序都会被病毒所感染，这使得该病毒的清除难度极大。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；关闭或删除系统中不需要的服务；很多病毒利用漏洞传播，一定要及时给系统打补丁；安装专业的防毒软件进行实时监控，平时上网的时候一定要打开防病毒软件的实时监控功能。
