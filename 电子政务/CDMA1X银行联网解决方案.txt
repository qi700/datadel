CDMA1X银行联网解决方案
1、需求分析1.1 CDMA1X网络及特点CDMA1X是一种新型的移动数据通信业务。CDMA1X是中国联通在完善优化CDMA网络建设过程中推出的第2.75代通信技术，它在目前的IS-95 CDMA
网络的基础上进一步升级了无线接口，使其支持高速补充业务信道，从而可实现高速互联网接入服务。CDMA1X在传递速率、功能方面较GPRS具有突出的技术优势， CDMA1X 理论速率153.6Kbps，目前实际测试速率为80kbps，超过目前普通有线电话拨号上网的56kbps，是平均30-40kbps的CDMA1X数据传输的2-3倍。特别适合像金融交易、远程监测等行业的通信需求，完全取代过去传统的有线MODEM、X.25、数传电台、短信等通信方式。CDMA1X通信方式更适合于银行联网业务，目前各省银行数据中心与各营业厅采用DDN 64K传送数据，费用不便宜。CDMA1X无线传输数据有以下优势：1）CDMA1X用户可随意分布和移动自己的网点，无需担心线路的维护或有线在移机时导致的通讯中断。建设新的营业厅无需进行拉线、埋线等工作。较光纤或专线系统投资较少，设备安装方便。2）终端价格比较低。与DDN相比，较DTU或DDN专线Modem其终端价格便宜很多。3）CDMA1X资费便宜，计费合理。CDMA1X 资费包月比有线电话网络资费还便宜。银行联网业务没有大数据量的信息传输，不必要采用资费很高的专线（DDN、帧中继）。CDMA1X还可根据通信的数据量和提供的服务质量进行计费。在CDMA1X网中，用户只需与网络建立一次连接，就可长时间的保持这种连接，并只在传输数据时才占用信道并被计费，保持时不占用信道不计费。这样，营业厅既不用频繁建立连接，也不必支付传输间隙时的费用。4）CDMA1X能最好地支持频繁的、少量突发型数据业务。通信质量稳定可靠，永不掉线。5）CDMA1X网络接入速度快，提供了与现有数据网的无缝连接。由于CDMA1X网本身就是一个分组型数据网, 支持TCP/IP、X.25协议，因此无需经过PSTN等网络的转接，直接与分组数据网（IP网或X.25网）互通，接入速度仅几秒钟，快于电路型数据业务。采用TCP/IP协议，较以前的无线数据网络（集群，双向传呼，GSM短信息）而言，网络接入更加直接方便。6）覆盖较好。比较很多无线数据网络（集群，双向传呼，CDPD）而言，其网络覆盖是最好的。

1.3 CDMA联网方案
图3 营业厅CDMA1X方式联网示意图
归属于不同基站（BTS）的营业厅终端设备，利用CDMA IP MODEM 连接到CDMA1X网络，在联通公司接入网关设备端为各营业厅终端分配一个CDMA1X内网的地址，此地址可以访问中心的服务器；所以以此类接入方式可以实现有线网络实现的互联功能。而在建设费用,建设时间,维护精力成本上比有线宽带方式有了很大的节约。另外，中心服务器不必走INTERNET，联网方式完全是在CDMA1X网络内部实现。利用中国联通提供的CDMA1X网络进行业务数据的传输，能够保证其安全性：1）物理链路的数据本身已经采用了加密传送；2）申请APN服务就像申请其他数据业务一样方便、快捷，同时拥有了数据专线的安全性。2. 我们的解决方案
2.1 CDMA1X IP Modem
图4 CDMA IP Modem功能示意图
CDMA1X IP MODEM是一款集成了CDMA1X MODEM 传输功能和TCP/IP网络、路由功能于一体的新型无线移动通信网络设备。广域网络采用CDMA1X，内部网络采用标准100BaseT 和串口（包括RS232、RS485 等）连接两种方式。根据不同的应用环境，物理上的这三个接口能够灵活运用，采用不同的连接方式，使该设备具备了三大功能：以太网→CDMA1X 网络的路由功能；串口→以太网网络转换功能；串口→CDMA1X 网络转换功能。以太网→CDMA1X网络的路由功能
连接内部局域网络（LAN）和CDMA1X 无线广域网络（WAN），能通过NAT技术，让内部的多台IP终端设备（如PC）以CDMA1X 无线方式同时共享接入互联网络/企业网络，使用现有的各种Internet 网络服务，如WEB、EMAIL、FTP等及企业各种应用。适用范围：内部网络已经是标准的TCP/IP网络，只是在广域网络接入方式上选择了无线移动CDMA1X网络。
图5 以太网→CDMA1X 网络的路由功能示意图
串口→CDMA1X网络转换功能
由于CDMA1X是基于TCP/IP协议的网络，要求接入的设备必须符合TCP/IP规范，而大多数传统数据终端设备（像金融POS、监测监控采集设备RTU等）均不具有IP功能，这些终端设备借助CDMA1X IP MODEM，能轻松方便地通过串口接入CDMA1X网络，实现远程无线联网。
图6 串口→CDMA1X 网络转换功能示意图
适用范围：金融POS、ATM、监测设备等带有传统串口的非IP终端设备通过CDMA1X实现无线联网。可以采用CDMA1X IP MODEM特有的透明传送方式，除了通信网络采用CDMA1X外，其他已有网络结构并不改变，充分利于过去存在的大量设备，快速经济无线联网。
串口→以太网网络转换功能（可去掉无线模块）
将传统的非IP终端通过RS232/RS485串口直接转换为具有标准TCP/IP 100BaseT接口的网络设备，解决了同一位置有多台串口终端需要同时通过CDMA1X联网的情况。
图7 串口→以太网网络转换功能示意图
CDMA1X IP MODEM 只能提供一个串口解决单台串口设备接入CDMA1X 问题，对同一地点有多台串口终端需要接入CDMA1X 的情况，如果每台设备均通过CDMA1X IP MODEM的串口→CDMA1X网络转换功能接入CDMA1X无线网络，干扰较大，并且总体成本方面投入较大。因此可以先采用串口→以太网网络转换功能将所有串口终端用局域网络的方式联网，最后集中采用一台CDMA1X IP MODEM的以太网→CDMA1X网络路由功能接入CDMA1X网络，整体方案具有很高的性能价格比。
