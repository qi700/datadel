Web Server解决方案
方案简述     当人类跨入新世纪的时候， 全世界都面临着新的商业模式的冲击， IT 系统和 Internet 逐渐成为所有核心业务的基础。 在这种环境下，企业需要一种平台， 一种可以适应企业规模快速发展， 业务不断扩展的可扩展性的WEB平台；一种能运行关键业务应用的稳定的 WEB平台；一种能高速处理业务，满足信息时代快速业务反应的WEB平台， 该平台就是对所有的信息及处理起到一个网络中心作用的网络操作系统和WEB服务器。     为满足新的信息处理和电子商务应用的需求， 降低企业成本，提高企业的竞争能力，现代企业正寻找这样一个企业级网络操作系统， 它具有高性能、高稳定性、高可靠性而且能提供低成本，使用开放的标准与技术，安装、管理和维护简便。这种网络操作系统的最佳选择就是遵循自由软件之路的 Linux 操作系统，Linux 也是目前增长速度最快的操作系统。     适用对象     各种大中小型企业，可发布公司信息     功能概述     Turbolinux Server 6.1 提供了所有的关键性系统服务，包括功能强大的多任务支持、客户机的无缝访问和高速连接，其提供的高速 TCP/IP 网络、Internet 邮件服务、全球最流行的 Apache Web 服务器、Java 、系统备份和恢复工具、 Windows 文件和打印服务， 可以给企业用户提供更大的自由空间和更灵活的解决方案。     方案优势     稳定的、最佳的高性能内核     在核心配置工具中支持非Intel 处理器     提供用户级工具 isapnp 来设置PnP 设备     可以将 IDE 子系统作为一个模块进行加载或卸载     支持并口 IDE 设备，允许串口设备共享中断，支持最新的 UART 芯片支持更多的文件系统和分区类型：Microsoft NTFS 和 Windows 98 FAT32 设备极强的可伸缩性和系统容量     系统支持的内存容量可达 4GB，单个服务器可以支持更多的用户和数据模型。     可以支持多达 32种文件系统，除ex2 文件系统，dos 文件系统之外， 系统还支持 /proc 文件系统。     系统可以支持多达32个CPU，文件系统最大可达2TB，交换分区可以大于128MB 的限制。     系统可以支持大量的存储设备：高速、高容量，有不同的类型和容量(IDE、ESDI 和SCSI 等)。支持固定和可拆除硬盘驱动器，支持多种工业标准的 RAID 硬件系统。     支持各种磁带设备，支持multisession CD支持I2O。    高可靠性、安全性和可用性     Turbolinux 以稳固的 2.2.17 版核心为基础，通过与外围的工具和实用程序完美的结合使系统更加可靠。     Turbolinux Server支持多种型号的RAID控制器，如:Mylex DAC960，AMI MegaRAID，Compaq SmartRAID，IBM ServerRAID等等。如果需要软件RAID，Turbolinux Server 6.1支持软件RAID0，RAID1，RAID4和RAID5。     Turbolinux Server 6.1 系统中收录了安全性应用中不可缺少的 SSH 和 SSL。WEB 服务器采用了 RSA 加密技术，为实现商务应用提供了安全的电子商务平台。     系统中减少了不必要的工具和X对话，增强了系统的稳定性和安全性。     另外，Turbolinux 6.1 包括所有最新的服务器和应用软件的安全补丁，我们时刻跟踪 BUGTRAQ 的报告，并且通过 FTP 提供最新的系统漏洞修补程序。     全方位支持业界领先的WEB服务     Turbolinux Server集成了业界领先的 Apache 服务器(http://www.apache.org)。 Apache 是互联网上第一品牌的网页服务器，在互联网上超过 60%以上的站点使用 Apache 服务器， 或由其衍生出来的产品。

成功案例     鲨威体坛的网站建设采用纯Linux解决方案，而且全线采用Turbolinux公司的产品，包括Turbolinux Server、Turbolinux Data Server、Turbolinux Cluster Server 和 Turbolinux Workstation等。     网站的网络采用高效并且易于扩充的三层结构，而且鲨威体坛的企业内部网也是由Turbolinux产品构建的。     第一层 Cache Server Cache Server采用了和Web Server完全不同的数据存贮方式和运行模式，因此对一段时间内频繁被访问的网页的响应速度有很大提高。     第二层 Web Servers 由于鲨威体坛拥有大量的实时动态内容（如新闻等），因此在Web服务 器上采用了PHP3从后台数据库中提取数据并生成动态网页。    第三层 Database Server 后台的Database Server是整个系统的核心，在这一层，鲨威体坛选用了Turbolinux Data Server plus Oracle产品。鲨威体坛提供的HA高可用性方案更是确保了整个网站的7×24小时不间断运行。
