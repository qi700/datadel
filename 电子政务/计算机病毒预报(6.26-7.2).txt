计算机病毒预报(6.26-7.2)
病毒名称：“灰鸽子变种jt”(Hack.Huigezi.jt)     危害程度：中高     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, Unix, Linux     病毒危害：     1. 在系统目录中生成病毒文件     2. 修改注册表特定项     3. 添加服务，留下后门     感染形式：     这是一个可进行远程控制的灰鸽子病毒又一变种。该病毒在系统目录中生成病毒文件并修改注册表特定项。该病毒在系统中添加命GrayPigeon_Hacker.com.cn的服务，在系统留下后门，随时给人入侵可能。但该病毒不会自动传播。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。          病毒名称：“索钱”（Hack.SnuHay.a）     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1. 通过邮件传播     2. 中止掉除指定进程     3. 寻找系统中文档文件，设置属性     4. 具有一定敲诈行为     感染形式：     这是一个隐藏用户文档的黑客程序。该病毒先枚举系统大量进程，如EXPLORER.EXE、SYSTRAY.EXE、HCOUNT.EXE、IRMON.EXE、MSTASK.EXE、SPOOL32.EXE等，并中止掉除以上进程以外所有进程。该病毒寻找系统中文档文件，并移到磁盘根目录下，将其文件夹设为隐藏与系统属性，该病毒具有一定敲诈行为。     预防和清除：     建立良好的安全习惯，关闭或删除系统中不需要的服务；很多病毒利用漏洞传播，一定要及时给系统打补丁；安装专业的防毒软件进行实时监控，平时上网的时候一定要打开防病毒软件的实时监控功能。          病毒名称：“恶鹰变种RZ”（Worm.Mail.Bagle.rz）     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1. 自动搜集电子邮件地址     2. 发送大量携带病毒的电子邮件     3. 中毒电脑可能出现运行变慢     4. 造成局域网速度变慢甚至阻塞     感染形式：     这是一个通过电子邮件传播的蠕虫病毒。该病毒会自动搜集电子邮件地址，向这些地址发送大量携带病毒的电子邮件。发送的病毒邮件主题为“01”、“02”、“05”或其它类似的数字。同时在病毒邮件中包含名为“07.zip”、“04.zip”、“02.zip”或其它类似数字名字的附件。附件压缩包中包含文件名为“16-06- 2006.exe”的木马病毒。中毒电脑可能出现运行变慢的情况，病毒邮件的泛滥还可能造成局域网速度变慢甚至阻塞。     预防和清除：     建立良好的安全习惯，关闭或删除系统中不需要的服务；很多病毒利用漏洞传播，一定要及时给系统打补丁；安装专业的防毒软件进行实时监控，平时上网的时候一定要打开防病毒软件的实时监控功能。
