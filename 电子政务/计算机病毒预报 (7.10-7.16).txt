计算机病毒预报 (7.10-7.16)
病毒名称：“银行大盗变种ic”(Win32.Troj.Banker.ic)     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, Unix, Linux     病毒危害：     1.修改注册表，实现开机自启     2.检测用户是否登录工商银行的网页     3.记录用户输入银行帐户号与密码     4.把信息发送送到指定的邮箱     5.给用户造成很大的损失     感染形式：     这是一个专门盗取工商银行帐号与密码的木马病毒。该病毒通过检测用户是否登录工商银行的网页，读取用户输入的帐户号码/用户名与密码，然后把信息发送到一定的邮箱中，使用户网上银行的帐户密码泄漏，并添加注册表键值，使病毒能随Windows启动。该病毒一但得到了用户的帐号与密码后，会很快地把用户帐户上的金钱转走，给用户造成很大的损失。     预防和清除：     建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。          病毒名称：“阿布特变种xv”（Win32.Hack.Rbot.xv）     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1.在系统中生成病毒文件     2.添加注册表启动项     3.其主进程会把原病毒文件删除     4.利用系统漏洞感染其他机器     感染形式：     这是一个可以秘密控制受感染机器的黑客程序。该病毒在系统目标中生成病毒文件，添加注册表启动项。该病毒具有蠕虫特征，会自动传播，运行该病毒后，其主进程会把原病毒文件删除掉，同时它还会主动连接其他机器端口，利用系统漏洞感染其他机器。     预防和清除：     建立良好的安全习惯，关闭或删除系统中不需要的服务；很多病毒利用漏洞传播，一定要及时给系统打补丁；安装专业的防毒软件进行实时监控，平时上网的时候一定要打开防病毒软件的实时监控功能。          病毒名称：“雏鹰变种ar”（Worm/BBEagle.ar）     危害程度：中     受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003     未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux     病毒危害：     1.利用自带的SMTP下群发带毒邮件     2.屏幕显示虚假错误信息     3.修改注册表，删除与安全相关的键值     4.侦听黑客指令，搜索邮箱地址     5.在共享目录下创建蠕虫副本     感染形式：     这是一个网络蠕虫，开启TCP 1234端口，利用自带的SMTP群发带毒邮件。该病毒运行后，屏幕显示虚假错误信息。修改注册表，删除与安全相关的键值。开启TCP 1234端口，侦听黑客指令，从被感染计算机上搜索有效邮箱地址，利用自带的SMTP群发带毒邮件。另外，该病毒还可在共享目录下创建蠕虫副本，实现网络共享传播。     预防和清除：     建立良好的安全习惯，关闭或删除系统中不需要的服务；很多病毒利用漏洞传播，一定要及时给系统打补丁；安装专业的防毒软件进行实时监控，平时上网的时候一定要打开防病毒软件的实时监控功能。
