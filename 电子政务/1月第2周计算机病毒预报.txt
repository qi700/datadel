1月第2周计算机病毒预报
下周计算机病毒预报（2007年1月8至2007年1月14日）
病毒名称：“Wevoo变种ke”(Hack.Wevoo.ke)危害程度：中受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003未受影响的系统: Windows 3.x, Macintosh, Unix, Linux病毒危害：1. 感染系统目录2. 下载广告程序3. 安装广告软件并不断弹出感染形式： 这是一个广告程序下载，病毒感染系统目录下的PvSec.dll文件，它根据网站上的广告程序版本号下载广告程序到用户的计算机上并且进行安装，安装后的广告软件会严重影响用户工作。预防和清除：　  建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。
病毒名称：“游戏密码大盗变种ad”（Troj.PswGame.ad） 危害程度：中低受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003未受影响的系统: Windows 3.x, Macintosh, UNIX, Linux病毒危害：1. 监控网游登陆窗口2. 获取用户输入的信息3. 发送到指定的邮箱4. 感染计算机的根目录感染形式： 这是一个网游的盗号木马，该木马运行时会监控众多网游登陆窗口，以获取用户输入的信息，把所窃取的敏感信息发送到固定的邮箱。该病毒还会在被感染计算机的每个盘的根目录创建一个AutoRun.inf文件，使用户双击盘符图标就会运行该病毒。预防和清除： 建议电脑用户不要随便运行来历不明的文件，以免中毒受害。平时上网的时候一定要打开防病毒软件的实时监控功能。
病毒名称：“敲诈者变种a”(Troj.QiaoZhaZhe.a)危害程度：中受影响的系统: Windows 95, Windows 98, Windows Me, Windows NT, Windows 2000, Windows XP, Windows Server 2003未受影响的系统: Windows 3.x, Macintosh, Unix, Linux病毒危害：1. 删除非系统目录的一切可删除的文件2. 弹出诱骗提示对话框3. 重启动计算机4. 给用户带来数据损失感染形式： 该病毒是一个“敲诈者”病毒，病毒运行后会删除非系统目录的一切可删除的文件，并弹出提示对话框：“警告：发现您硬盘内有盗版了的本公司软件，所以我们已将您硬盘内部份数据移位到了锁定的扇区，如要解开被锁的扇区将文件释放出来,请电邮至webmasters7@yahoo.com.cn购买相关的解锁软件”。当点击确定后会重启动计算机。用户中毒后千万不要乱动磁盘，建议使用“专用数据安全软件”对文件进行恢复。恢复时需要另一块新的硬盘，将数据恢复到新盘上面，以免覆盖其他没有被恢复的数据。预防和清除：建立良好的安全习惯，不打开可疑邮件和可疑网站；建议亲爱的电脑用户不要随便运行不知名程序或打开陌生人邮件的附件，以免中毒受损。
