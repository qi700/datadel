哈尔滨市作风建设活动的实施方案（要点）
按照省委的统一部署，市委决定，以县、处级以上领导班子和领导干部为重点，在全市开展以“讲党性、树新风、优环境、促发展”为主题的加强作风建设活动。
一、指导思想和目标要求
（一）指导思想
以“讲党性、树新风、优环境、促发展”为主题，以加强党性修养、树立弘扬优良作风为核心，以加强行政效能建设、优化发展环境为重点，以促进经济社会更好更快发展为根本目标和衡量标准，以“整顿机关作风，提高工作效率”年活动为载体，着力解决党性党风党纪方面群众反映强烈的突出问题和制约科学发展、影响行政效能、破坏发展环境方面的问题，进一步改进作风、提高能力、健全机制、形成氛围，为促进全市经济社会更好更快发展提供坚强保证。
（二）目标要求
1. 讲党性。要着力增强宗旨观念，切实做到立党为公、执政为民；着力提高实践能力，切实用党的科学理论指导工作实践；着力强化责任意识，切实履行党和人民赋予的职责；着力树立正确政绩观，切实按照客观规律谋划发展；着力树立正确利益观，切实把人民利益放在首位；着力增强党的纪律观念，切实维护党的团结统一。
2. 树新风。要在全市党员领导干部中大兴“四风”：一是大兴解放思想之风。二是大兴求真务实之风。三是大兴亲民为民之风。四是大兴廉洁从政之风。
3. 优环境。一要进一步减少和规范行政审批事项，压缩审批时限。二要加强电子政务和电子监察系统建设。三要加强对《行政许可法》贯彻执行情况的监督检查，建立和完善规范权力运行的监督管理制度。四要加强社会公共服务部门的行风建设，杜绝乱收费乱罚款、搞强制服务、设置服务陷阱等与民争利的问题。
4. 促发展。要以昂扬的状态思发展、以创新的能力谋发展、以扎实的作风抓发展、以问题的解决促发展、以正确的标准检验发展。确保省、市委确定的经济社会发展目标和任务实现。
二、主要任务及责任分工
（一）强化思想教育
1. 开展政治理论学习教育。组织党员领导干部认真学习党章和中国特色社会主义理论体系。对县处级以上领导干部进行一次以学习胡锦涛总书记关于加强党性修养、树立和弘扬良好作风的重要论述为主要内容的集中培训。（牵头部门：市委宣传部，责任部门：市委组织部、市委党校）
2. 开展马克思主义世界观教育。将党性教育和马克思主义世界观教育纳入党校今后各类主体班次的教学计划，引导广大党员干部牢固树立正确的世界观、人生观、价值观、权力观、地位观、利益观，特别是胡锦涛总书记在中央纪委十七届三次全会上提出的“新三观”，即事业观、工作观、政绩观，将“新三观”落实到谋划工作中，落实到推动科学发展中。（牵头部门：市委组织部，责任部门：市委宣传部、市委党校）
3. 开展正面典型教育。选树一批党性强、作风硬、能干事、干实事的先进典型，组织召开事迹报告会和巡回演讲，教育引导广大党员干部树立和弘扬良好作风。（牵头部门：市委组织部，责任部门：市纪委、市委宣传部）
4. 开展警示教育。抓住几起作风建设方面的反面典型，对广大党员干部进行警示教育，警示、促进各级党员干部尤其是领导干部强化宗旨意识、提升道德素养和转变工作作风。（牵头部门：市纪委，责任部门：市委组织部、市委宣传部）
5. 开展光荣传统教育。认真贯彻落实省委关于新时期深入学习和弘扬大庆精神的决定，教育广大党员干部继承光荣传统，弘扬优良作风，带头艰苦奋斗、勤俭节约，努力降低行政运行成本，确保今年因公出国（境）经费要在近3年平均数的基础上压缩20%，车辆购置及运行费用支出要在近3年平均数基础上降低15%，公务费用接待支出要在去年基础上削减10%，节电、节油、节水指标要在去年基础上降低5%的目标落到实处。把有限的财力用在发展上。（牵头部门：市委宣传部，责任部门：市委组织部、市级各新闻媒体、市委党校）
（二）强化制度落实
1. 认真落实联系群众制度和调查研究制度。在继续深入开展“万名干部进农村、服务农民促发展”和“千名干部下基层帮企业保增长”活动的同时，围绕解难题、抓落实、保增长，从4月中旬开始，组织各级领导干部大下基层，研究解决影响和制约科学发展的重点难点问题，保证市委确定的经济社会发展目标如期实现。（牵头部门：市委组织部，责任部门：市委办公厅、市政府办公厅、市直机关工委）
2. 认真落实党员领导干部个人有关事项报告制度。加强对报告事项的监督检查，及时纠正和解决存在的问题。组织县以上领导班子召开一次专题民主生活会，组织区、县（市）委班子开展好届中述职述廉。（牵头部门：市委组织部，责任部门：市纪委）
3. 严格执行领导干部诫勉谈话制度。对政治思想、履行职责、工作作风、道德品质、勤政廉政等方面存在苗头性问题的领导干部及时进行诫勉谈话。实施地方党委委员、纪委委员党内询问和质询办法。（牵头部门：市纪委，责任部门：市委组织部）
4. 积极推进党务公开，研究制定哈尔滨市推进党务公开工作指导意见。（牵头部门：市委办公厅，责任部门：市纪委、市委组织部、市委宣传部）
5. 加强对领导干部经济责任审计，认真纠正违反财经纪律的行为。（牵头部门：市审计局，责任部门：市纪委、市委组织部）
6. 探索监督保护党员干部干事创业的基本模式，关口前移、预警护航，逐步建立支持改革者、鼓励干事者、追究诬陷者的机制、制度。（牵头部门：市纪委，责任部门：市委组织部、市人事局）
（三）加强机关效能建设
1. 加快推进政府部门机构改革，7月底前，市直部门初步构建起覆盖所有行政和司法权力的制度体系。（牵头部门：市编办，责任部门：市政府办公厅、市监察局、市司法局、市政府法制办）
2. 按照“坚持、完善、废止”的要求，对现有制度进行一次全面系统的清理和审查，规范权力运行。（牵头部门：市政府办公厅，责任部门：市监察局、市政府法制办）
3. 进一步减少和规范行政审批事项，压缩审批时限。市直各部门非行政许可审批事项再缩减20%左右，单项行政审批时间在法定审批时限基础上再缩短20%左右，继续缩小投资审核范围，除按照法定的投资审核规定进行审核外，取消部门自定的审核条件。（牵头部门：市编办，责任部门：市发改委、市政府办公厅、市监察局、市人事局、市商务局、市政府法制办、市行政审批中心）
4. 加强电子政务系统建设，推广南岗区建立电子监察系统对行政审批流程、“一口受理”服务环节和机关办公秩序实行网上全程监控的做法，建立统一的网上行政审批平台，实行网上申报、网上审批、网上招投标。开展行政审批电子监察试点工作，年内将产权交易、政府采购、经营性土地招拍挂、建筑工程招投标四个要素市场纳入平台统一管理，切实提高行政部门的执行力和公信力。（牵头部门：市政府办公厅，责任部门：市监察局、市财政局、市编办、市行政审批中心）
5. 加强对《行政许可法》贯彻执行情况的监督检查。加强对环保项目评审、安全生产监管、建设工程管理、土地和矿产资源管理、食品药品审批、评审的监督，重点检查行政许可的实施和评后的监管环节，纠正重审批轻监管等问题。（牵头部门：市监察局，责任部门：市政府办公厅、市政府法制办、市纠风办）
6. 大力推行政务公开和公共服务部门办事公开，建立健全政府信息发布制度，促进行政权力公开透明运行。（牵头部门：市政府办公厅，责任部门：市直机关工委、市监察局、市人事局、市政府法制办）
7. 全面落实首问负责制、收费公示制、限时办结制、服务承诺制、一次性告知制、投诉问责制及AB角等制度，促使廉政建设和政风建设进一步加强。（牵头部门：市监察局，责任部门：市政府办公厅、市人事局、市纠风办、市编办、市政府法制办）
8. 加强社会公共服务部门的行风建设。深入开展民主评议行风活动，继续办好“行风热线”和“纠风直通车”，发挥行风监督员队伍作用。（牵头部门：市纠风办、市广播电视局，责任部门：市直相关部门）
9. 开展行政绩效管理试点工作。（牵头部门：市政府法制办，责任部门：市委组织部、市直机关工委、市人事局、市监察局）
10. 组建市效能投诉中心，建立与省及各区、县（市）上下联动的效能监察组织保障体系。在《哈尔滨市行政机关领导问责暂行办法》基础上，制定实施《行政机关工作人员行政过错追究暂行办法》，严肃查处拖拉梗阻、刁难勒卡、乱收乱罚、违法行政等行为。严肃追究决策失误、治理失当、政令失畅、行为失范等行政责任及连带责任。对严重阻碍发展、破坏经济环境的违纪违法案件严肃处理。（牵头部门：市监察局，责任部门：市直机关工委、市政府办公厅、市人事局、市财政局、市商务局、市编办、市政府法制办）
（四）创建良好的舆论环境
要在市、县（市）级各主要新闻媒体上通过开办专栏、专题节目、公益广告、系列报道、评论员文章等多种形式，为开展加强作风建设活动营造浓厚的舆论氛围。加强舆论监督，及时报道总结开展作风建设活动的好做法、好典型；对一些作风方面存在问题的反面典型进行剖析、曝光。（牵头部门：市委宣传部，责任部门：市纪委、市委组织部、市级各新闻媒体）
（五）强化督促检查
1. 充分发挥人大、政协、行政、司法、群众、社会的监督作用，通过党代会代表、人大代表和政协委员视察、质询听证和办理提案、聘请党风政风机关作风监督员、民主评议行风、召开新闻发布会和情况通报会等形式，构建全方位、多层次的作风监督网络，增强监督合力和实效。（牵头部门：市纪委，责任部门：市委办公厅、市人大办公厅、市政府办公厅、市政协办公厅、市委宣传部）
2. 采取党风巡视等多种形式开展加强作风建设活动情况的监督检查，及时发现和纠正存在的党性不强、作风不正以及其他党风政风方面的苗头性、倾向性和违反哈尔滨市党政机关工作纪律《八条规定》的问题。（牵头部门：市纪委，责任部门：市委办公厅、市委组织部、市直机关工委、市政府办公厅、市学习实践科学发展观领导小组办公室）
3. 组织部分市直部门（单位），通过电视、报纸等新闻媒体，由主要领导公开向社会作出加强作风建设承诺，主动接受公众监督。（牵头部门：市直机关工委，配合部门：市委宣传部）
4. 探索推广重点工程项目跟进监督保障模式，协调各方、惩防并举，确保重点工程项目顺利实施。（牵头部门：市纪委，责任部门：市委组织部、市编办）
三、保证措施及要求
（一）高度重视，摆上日程
各地、各部门要把作风建设活动作为当前重大政治任务，加强组织领导，摆上工作日程，纳入总体工作，全力抓好推进，保证加强作风建设活动有组织、有计划的开展，抓出成效。
（二）明确任务，落实责任
1. 各项任务的牵头部门要按照市委开展作风建设活动的要求对所承担工作任务制定推进落实工作方案，明确具体内容、时限要求和推进措施，做到任务目标明确，措施切实可行，责任落实到人。
2. 市学习实践科学发展观活动领导小组办公室负责抓好区、县（市）的作风建设活动；市直机关工委负责抓好市直机关的作风建设活动。要加强组织领导和工作指导，搞好情况综合，及时总结推广好经验、好做法，推动工作开展。
3. 各级领导机关和领导干部要以身作则、率先垂范。主要领导干部要认真履行第一责任人的职责，其他班子成员要各负其责，认真抓好分管战线和部门的作风建设活动。市委作风建设活动推进组将适时派出工作组，对各地、各部门开展作风建设活动情况进行监督检查，检查结果通报全市，并纳入目标责任制考核和党风廉政建设责任制考核之中。
4. 各牵头部门和市直机关工委、市学习实践科学发展观活动领导小组办公室每一季度要对工作开展情况进行一次总结，年底进行一次全面检查和总结，并于4月20日前将负责此项工作的组织机构和人员名单、联系电话及推进落实工作方案报市作风建设活动领导小组办公室（联系人：王艳，传真电话：84568536，电子邮箱：DFS7777@163.COM）。
（三）协调配合，形成合力。各级党委要对作风建设活动负总责。各牵头部门要积极发挥组织协调作用，责任部门要增强配合意识，形成各负其责、齐抓共管的工作格局，保证加强作风建设活动各项任务落到实处。
（四）统筹推进，务求实效。作风建设要与深入学习实践科学发展观活动紧密结合、与“项目建设年”活动紧密结合、与“整顿机关作风、提高工作效率”活动紧密结合，相互促进、共同提高，把作风建设活动不断引向深入。
四、工作推进机构
成立市委作风建设活动推进领导小组。推进领导小组下设办公室，设在市纪委，负责组织协调、综合反馈和督促检查。
