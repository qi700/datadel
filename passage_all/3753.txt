无锡审计局的“e”审之路更流畅
无锡审计局的“e”审之路更流畅,政府,“金审工程”建成后，作为国家监督机构--审计署机关以及驻地方的18个审计特派员办事处、30余个省级审计机关形成城际广域联接，将依托国家统一电子政务网络平台。目前全国审计系统已购置了两万余台服务器、计算机用于,北大方正集团"
近年来，随着我国加入WTO，政府的电子政务工程加速，我国的财政、金融、税务、海关等领域信息化进程迅速推进，政府部门、国有企业等被审计单位的会计核算、财务管理、经营管理电子化日益普及，审计信息化应运而生，所以努力提高审计管理信息化水平已成为我国审计行业亟待解决的问题。实现审计信息化的跨越式发展，使之符合国家号召的"电子政务"信息化发展的需求，已是摆在我们面前的十分现实的奋斗目标，也是中国审计信息化发展的必然趋势。
“金审工程”是第一个由国家计委立项，完全由国家财政拔款支持的电子政务项目。“金审工程”建成后，作为国家监督机构--审计署机关以及驻地方的18个审计特派员办事处、30余个省级审计机关形成城际广域联接，将依托国家统一电子政务网络平台。目前全国审计系统已购置了两万余台服务器、计算机用于部门办公、文字处理、现场审计和部分信息发布，已有一批审计软件应用于审计作业和机关日常事务管理。
数据安全，重中之重
无锡审计局是无锡地区负责审计工作的单位，其设置有法制综合处、财政金融审计处、行政事业审计处、经贸审计处和外资审计处等多个审计机构。无锡审计局是国内较早建立审计工作专用计算机网络工作平台的审计部门，在国家“金审工程”的指导下，无锡审计局已经建成基于计算机应用平台、实现数据共享的办公和业务应用系统。
目前无锡审计局内部网络中有一台主要的业务系统服务器，该服务器配置有Windows 2000 的AD，主要完成网络中用户的管理工作，安装有内部办公的审计系统及审计系统的后台数据库 SQL Server 2000，该服务器成为审计局日常办公的平台。
由于业务系统对于无锡审计局来说是他们内部办公的主要系统，再加上审计系统业务的特殊性，对系统数据的安全性、准确性、完整性要求非常高，需要保证系统在数据的采集、传输和存储过程中，是可靠的。所以对于无锡审计局来说，数据的安全是所有工作的重中之重，所以用户希望数据安全解决方案满足以下需求：
1、由于目前无锡审计局是在服务器本机上备份数据，这样无法避免本机硬盘故障、其它 系统故障、机房故障而导致的数据丢失。所以用户需要对业务系统中的SQL Server 数据库进行全面的备份。同时要求无论是业务系统还是数据出现问题以后，都可以进行成功的恢复。
2、由于目前无锡审计局的数据备份放在管理控制端服务器上，但管理端属于公共性平台，数据安全受到威胁。所以用户希望数据安全解决方案可以对业务系统服务器上AD的相关数据进行保护。
3、由于目前无锡审计局的数据备份采用人工手动的方式进行备份，这不仅加大了IT管理人员的工作量，同时由于误操作等人为原因，使数据的不安全因素大为提高。因此，用户需要建立一个自动的备份环境，备份不需要太多的人工干预，可以按照自己设置的日程自动执行。
4、由于具有大量的审计数据的信息量，所以无锡审计局要求服务器的存储空间能达到640GB，主要用于存储业务系统、数据和档案文件。
数据可靠，坚如磐石
审计数据的重要性不言而喻，系统数据一旦受到破坏，则对无锡审计局的审计管理工作造成极大的损失。那么，如何为无锡审计局的审计数据建立一个强大的保护盾牌呢？方正科技仔细研究客户的需求后，从无锡审计局系统的需求实际情况出发，为无锡审计局提出了基于方正科技SecuStor存储服务器的数据备份解决方案。
方正科技根据客户实际需求制定的解决方案实施后，提高了无锡审计局审计应用和效率，赋予无锡审计局更大的发展空间：
1、现在无锡审计局的业务系统已经实现在设定的时间内将数据库自动定时备份至相应的存储空间，同时按照文件夹附带日期的方式来区别某天的数据备份,大大的提高了无锡审计局的工作效率。
2、现在无锡审计局的整个备份过程1分钟之内即可完成，备份速度和恢复速度较快，相对以前的方式极大地提高了备份效率。
3、现在无锡审计局的数据备份工作已经实现整个备份过程完全自动化，不需任何人为因素干预，降低了系统管理员的工作繁杂程度，而且进一步提高了数据安全。
毫无疑问，方正科技为无锡审计局提供的这套数据安全解决方案提高了用户的整体信息化应用水平，利用信息化手段实现了用户在一个平台上实现更加科学、合理、严密、高效的管理，为无锡审计局的集中工作和管理带来了前所未有的便利，极大的提高了他们的工作效率。方正科技作为国内实力雄厚的行业解决方案提供商，积累了丰富的行业信息化经验，这次和无锡审计局的愉快合作，为无锡审计局乃至全国的“金税工程”建设贡献了一分力量。
