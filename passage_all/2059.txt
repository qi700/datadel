嘉定区提升一体化电子政务平台效率
上海嘉定区位于上海西北部，东与宝山、普陀两区接壤；西与江苏省昆山市毗连；南襟吴淞江（苏州河），与闵行、长宁、青浦三区相望；北依浏河，与江苏省太仓市为邻。总面积 458.8 平方公里。行政区域界线总长 167.8461 公里。全境地势平坦，东北略高，西南稍低。市、区级河道藻浜、练祁塘、娄塘河横卧东西，向东流经宝山区直通长江和黄浦江；盐铁塘、横沥、新槎浦纵贯南北，与吴淞江、浏河相连。
业务状况：
上海市嘉定区信息化委员会承担着电子政务服务的责任，在过去几年积累的基础之上，嘉定区新一代电子政务平台按照“一体化”思路，基于完善的信息基础设施体系，以统一的用户管理与授权、多渠道通信平台、应用支撑平台和灵活的协同工作门户，打造灵活、高效、人性化的信息化工作环境，促进政府职能转变，提升社会管理和公共服务效能。该平台2007年4 份正式启用以来，在全区各级政府部门得到快速普及和深入应用。
解决方案：
Office Communications Server 2007     Office Communicator 2007     Exchange Server 2007     Microsoft Office Outlook 2007
收益：
人性化的功能设计    丰富的通讯设备支持能力     即时的会议能力
“首先Live Communications Server 2005 让我们感受到即时沟通系统对政务日常工作的切实作用，而 Office Communications Server 2007 更为人性化功能设计及微软及合作伙伴强大的技术支持，使我们相信使用微软统一沟通解决方案能让沟通更方便、工作开展更快捷。”上海市嘉定区信息中心赵文晓主任如是说。
项目背景
上海市嘉定区信息化委员会在过去几年积累的基础之上，嘉定区基于Microsoft .Net的新一代电子政务平台按照“一体化”思路，基于完善的信息基础设施体系，以统一的用户管理与授权、多渠道通信平台、应用支撑平台和灵活的协同工作门户，该平台2007年4月份正式启用以来，在全区各级政府部门得到快速普及和深入应用。应用主要由统一办公门户、统一授权系统、办公信息系统、统一消息服务、统一邮件管理等组成。
其中统一消息服务、统一邮件管理是构建于Live Communications Server 2005和Exchange Server 2003的基础之上，其中统一消息管理系统解决了多应用系统各种时效性消息的统一集成展现问题，以及实现了点对点之间多方式的即时通讯，真正形成了一个信息统一展现与交互的平台，从而实现了资源共享和协同办公。
此外系统还开发了包括登陆提醒、及时提醒、交互机器人、与Office 系统集成、与门户集成、与业务系统集成、与办公系统的集成等特色功能。
